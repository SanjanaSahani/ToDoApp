# React Todo App

A simple todo application built with React.js and Vite.

## Features

- Add , edit and remove tasks
- Mark tasks as complete
- Filter tasks based on status (All, Active, Completed)

## Technologies Used

- React.js
- Vite

## Getting Started

### installation

install - node.js
npm create vite@latest
npm install
cd React-ToDoApp
npm run dev
